# 🐾 AdotaPet

Plataforma web para **adoção de animais domésticos**, conectando pessoas que desejam adotar com aquelas que querem doar pets de forma simples, rápida e segura.

---

##  Sobre o Projeto

O **Adopet** foi desenvolvido com o objetivo de facilitar o processo de adoção responsável de animais, oferecendo uma plataforma onde usuários podem:

* 🐶 Cadastrar animais para adoção
* 🔍 Buscar pets com base em preferências
* 💬 Entrar em contato diretamente com o responsável
* ⭐ Favoritar animais de interesse
